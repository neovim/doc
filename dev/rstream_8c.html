void rbuffer_consumed ( RBuffer *  rbuffer,
size_t  count 
)

Advances rbuffer read pointers to consume data. If the associated RStream had stopped because the buffer was full, this will restart it.

This is called automatically by rbuffer_read, but when using rbuffer_read_ptr directly, this needs to called after the data was consumed.
