Type of function called when the Stream buffer is filled with data

Parameters
streamThe Stream instance
rbufferThe associated RBuffer instance
countNumber of bytes to read. This must be respected if keeping the order of events is a requirement. This is because events may be queued and only processed later when more data is copied into to the buffer, so one read may starve another.
dataUser-defined data
eofIf the stream reached EOF.
