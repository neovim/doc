36 /*
37  * for each (previously) used block in the memfile there is one block header.
38  *
39  * The block may be linked in the used list OR in the free list.
40  * The used blocks are also kept in hash lists.
41  *
42  * The used list is a doubly linked list, most recently used block first.
43  * The blocks in the used list have a block of memory allocated.
44  * mf_used_count is the number of pages in the used list.
45  * The hash lists are used to quickly find a block in the used list.
46  * The free list is a single linked list, not sorted.
47  * The blocks in the free list have no block of memory allocated and
48  * the contents of the block in the file (if any) is irrelevant.
49  */
