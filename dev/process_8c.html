Synchronously wait for a process to finish

Parameters
processThe Process instance
msNumber of milliseconds to wait, 0 for not waiting, -1 for waiting until the process quits.
Returns
returns the status code of the exited process. -1 if the process is still running and the timeout has expired. Note that this is indistinguishable from the process returning -1 by itself. Which is possible on some OS. Returns -2 if an user has interruped the wait.
