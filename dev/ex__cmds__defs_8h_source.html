20 /*
21  * When adding an Ex command:
22  * 1. Add an entry to the table in src/nvim/ex_cmds.lua. Keep it sorted on the
23  * shortest version of the command name that works. If it doesn't start with
24  * a lower case letter, add it at the end.
25  *
26  * Each table entry is a table with the following keys:
27  *
28  * Key | Description
29  * ------- | -------------------------------------------------------------
30  * command | Name of the command. Required.
31  * enum | Name of the enum entry. If not set defaults to CMD_{command}.
32  * flags | A set of the flags from below list joined by bitwise or.
33  * func | Name of the function containing the implementation.
34  *
35  * Referenced function should be either non-static one or defined in
36  * ex_docmd.c and be coercible to ex_func_T type from below.
37  *
38  * All keys not described in the above table are reserved for future use.
39  *
40  * 2. Add a "case: CMD_xxx" in the big switch in ex_docmd.c.
41  * 3. Add an entry in the index for Ex commands at ":help ex-cmd-index".
42  * 4. Add documentation in ../doc/xxx.txt. Add a tag for both the short and
43  * long name of the command.
44  */
