682// This snprintf() only supports the following conversion specifiers:
683// s, c, b, B, d, u, o, x, X, p (and synonyms: i, D, U, O - see below)
684// with flags: '-', '+', ' ', '0' and '#'.
685// An asterisk is supported for field width as well as precision.

702// are discarded. If "str_m" is greater than zero it is guaranteed
703// the resulting string will be NUL-terminated.
