Documentation

The following resources are available online for users and developers of Neovim:

Users

Installation Instructions

Neovim is currently under development, and no releases have been published yet. To install a pre-alpha, experimental version of Neovim, please refer to the installation instructions in the Neovim Wiki.

User Manual

If you have installed Neovim, execute the :help command to access its user manual. For an overview of features specific to Neovim, see the introduction to Nvim (:help nvim-intro). This also contains instructions on how to enable support for Python plugins (:help nvim-python) such as YouCompleteMe.

Other helpful resources for both Neovim and Vim are the quick reference guide (:help quickref) and the Vim FAQ.

Troubleshooting

Lots of information is available in the Neovim Wiki. In particular, refer to the FAQ and Troubleshooting pages if you encounter problems. Breaking changes in unstable versions of Neovim are listed on Following HEAD.

If you're stuck, you can reach out to the Neovim community for help.

Developers & Contributors

Neovim Wiki

Neovim's wiki contains a number of links, hints and guides for Neovim users and developers.

Developer Documentation

Visit the developer documentation page for an up-to-date Doxygen documentation of Neovim's code.

Clang Report

Clang's static analyzer is executed periodically on the Neovim codebase. A list of current issues is published in the static analysis report.

Coverity Report

A list of issues found by Coverity is available at the Neovim Coverity project. To get access, have a look at the hints on the Neovim wiki.

Translation Report

An overview of the current status of translations in Neovim is published on the translation report page.
